# 📦 Proyecto 5 -Inventario básico

## 🎯 Objetivo
Registrar múltiples entradas con control de salida.

## 🧾 Enunciado
Crear un sistema que permita ingresar productos hasta escribir "fin".

## 📌 Requisitos
- Pedir nombres de productos
- Contar cuántos se ingresaron
- Terminar con "fin"

## ✅ Criterio de éxito
Se muestra la cantidad total de productos ingresados.
