# 📈 Proyecto 3 - Control de presupuesto

## 🎯 Objetivo
Combinar `while` + `if`.

## 🧾 Enunciado
Crear un sistema que acumule gastos y avise si se supera un límite.

## 📌 Requisitos
- Pedir gastos en bucle
- Sumar total
- Si total > 5000 → mostrar alerta
- Permitir continuar o salir

## ✅ Criterio de éxito
El sistema alerta correctamente cuando se supera el presupuesto.
