# 🔄 Ejercicios Resueltos — Conversión de Tipos en Python

---

## Ejercicio 1 — Convertir edad a int

**Enunciado:** Pedile al usuario su edad y convertila a número entero.

```python
edad = int(input("Ingresá tu edad: "))
print(edad)
```

**Ejemplo de ejecución:**
```
Ingresá tu edad: 25
25
```

---

## Ejercicio 2 — Sumar 5 a la edad

**Enunciado:** Pedile al usuario su edad, convertila a entero y mostrá cuántos años tendrá en 5 años.

```python
edad = int(input("Ingresá tu edad: "))
print("En 5 años tendrás:", edad + 5)
```

**Ejemplo de ejecución:**
```
Ingresá tu edad: 25
En 5 años tendrás: 30
```

## Ejercicio 4 — Multiplicar número

**Enunciado:** Pedile al usuario un número entero, convertilo y multiplicalo por `3`.

```python

```

**Ejemplo de ejecución:**

## Ejercicio 6 — Restar números

