# 🟡 Ejercicios Intermedios – Python (Input + Operaciones)

Practicá entrada de datos, operaciones y lógica básica 💻

---

## 📌 Instrucciones

* Usá `input()` para pedir datos
* Convertí cuando sea necesario (`int()`, `float()`)
* Mostrá resultados con `print()`

---

## 🧪 Ejercicios

Pedir nombre y mostrar saludo

```python

```
