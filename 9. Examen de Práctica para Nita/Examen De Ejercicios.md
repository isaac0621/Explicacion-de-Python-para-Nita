# 📗 Examen 2 – Ejercicios Separados

## 🎯 Objetivo
Resolver varios problemas prácticos.

---

## 🧾 Ejercicio 1 – Control de acceso

Enunciado:
Pedir edad y mostrar si puede ingresar.

Debe usar:
- input
- conversión
- if

Pista:
edad >= 18

---

## 🧾 Ejercicio 2 – Cálculo de compra

Enunciado:
Pedir precio y cantidad, mostrar total.

Debe usar:
- input
- float
- multiplicación

Pista:
total = precio * cantidad

---

## 🧾 Ejercicio 3 – Validación de datos

Enunciado:
Pedir edad hasta que sea mayor a 0.

Debe usar:
- while
- validación

Pista:
repetir si edad <= 0

---

## 🧾 Ejercicio 4 – Sistema repetitivo

Enunciado:
Pedir números hasta que el usuario escriba 0.

Debe usar:
- while
- break

Pista:
if numero == 0 → break

## 🧾 Ejercicio 5 – Clasificación

Enunciado:

- if / elif / else

## 🧾 Ejercicio 6 – Acumulador

## 🧾 Ejercicio 7 – Comparación

- if
