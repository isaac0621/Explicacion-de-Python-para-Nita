# 📘 Examen 1 – Caso Completo

## 🎯 Objetivo
Aplicar TODO en un solo problema:
- Variables
- Tipos de datos
- Input
- Conversión
- Condicionales
- Bucles
- while True

---

## 🧾 Enunciado

Trabajás en una empresa que vende productos.

Necesitás crear un sistema que:

1. Permita ingresar múltiples ventas
2. Cada venta tiene:
   - precio del producto
   - cantidad
3. Calcular total por venta
4. Acumular total general

---

## 📌 Reglas del negocio

- Si una venta supera 5000 → mostrar "Venta alta"
- Si no → "Venta normal"

---

## 🔁 Flujo

- El sistema debe repetirse hasta que el usuario escriba "no"
- Usar `while True`
- Usar `break` para salir

---

## 📤 Salida final

Mostrar:

- Total acumulado
- Cantidad de ventas realizadas
- Promedio por venta

---

## ⚠️ Validaciones

- No permitir números negativos
- Repetir si el dato es inválido

---

## 🧠 Pistas

- Usar acumulador → total += algo  
- Usar contador → contador += 1  
- Usar conversión → int() o float()  

---

## ✅ Criterio de éxito

- Funciona sin errores
- Usa todos los conceptos
- Permite múltiples ventas
