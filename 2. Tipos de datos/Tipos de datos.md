
## 2. 🔢 Tipos de datos

# Variable | Para que funciona | Ejemplo #
* 📝 String → Se utiliza para textos → "Hola"
* 🔢 Int → Se utiliza para número enteros → 31
* 📏 Float → Se utiliza para números con décimales → 1.65
* ✅ Bool → Se utiliza para decisiones (Verdadero/Falso) → True

```python
nombre = "Nita"
edad = 31
altura = 1.65
es_tica = True
```
